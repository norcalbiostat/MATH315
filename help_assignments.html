Frequently Asked Questions

Last Updated 2018-09-14 17:02:37

How do I complete the assignments?

  • Most homework assignments will come with a R markdown (*.Rmd) template file to use.
    • If you left click on the link and it opens in a HTML browser window, it will not work this way.
    • Right click on the link to download this file and save as or save target as.
  • Write your code and answers directly into the Rmd file. Spellcheck and knit to PDF before submission.
  • Links to the templates can be found in the homework assignment itself. A list of all assignment files can be found on the notes page.

How/Where do I turn in Homework?

Standard Assignments

Unless explicitly stated otherwise,

  • The file should be clearly named using this format: assignment_userid.pdf. E.g. univ_graphics_rdonatello.pdf
  • It should be in PDF format.
  • Uploaded to the corresponding assignment in Blackboard learn.

Project Assignments

  • Write the assignment as a team (analysis pair), directly in Google Docs. Due dates are typically Sunday.
  • During Mon & Tue, your assignment will be reviewed by 2 of your peers.
  • During Wed & Thu, you have the option of revising your assignment.
  • By Friday you should drop a note in the corresponding assignment in Blackboard Learn saying your assignment is ready for review.
    • You can upload to Blackboard learn the revised assignment, OR you can simply say “Done, see google drive”

Project assignments should be considered quality work and such refined, reviewed, and absolutely spellchecked before submission.

What are you looking for when you grade?

  • Most assignments include model examples directly in the assignment file itself.
  • Correct english grammer, spelling, formatting.
  • Concice, clear and complete answers.
  • Correct statistical methods used
  • That you answered the question posed.
  • No extreneous output or code
